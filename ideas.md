# Wind Turbine Calculator — Design Ideas

## Approach 1: Industrial Precision Dashboard
<response>
<text>
**Design Movement:** Industrial Data Visualization / Engineering Dashboard

**Core Principles:**
- Data-first layout with dense information hierarchy
- Monochromatic steel palette with amber/orange accent for critical values
- Grid-based technical aesthetic reminiscent of engineering schematics
- High-contrast readability for field use

**Color Philosophy:** Deep charcoal backgrounds (#1a1f2e) with steel-blue panels, amber (#f59e0b) for key metrics, and white text. Evokes precision instruments and control rooms.

**Layout Paradigm:** Full-width split layout — left panel for inputs (30%), right panel for live results (70%). Sidebar-style navigation for multi-step flow.

**Signature Elements:**
- Monospace font for numeric outputs
- Thin horizontal rule separators between data sections
- Glowing amber indicator dots for active fields

**Interaction Philosophy:** Every input change triggers immediate recalculation with animated counter transitions. Feels like a live instrument panel.

**Animation:** Number counters animate on change. Progress bars fill smoothly. Subtle pulse on "calculating" state.

**Typography System:** IBM Plex Mono for numbers/data, IBM Plex Sans for labels. Bold weight for results, regular for inputs.

## Approach 2: Clean Scientific Tool (SELECTED)
<response>
<text>
**Design Movement:** Scientific / Technical SaaS — Bauhaus-inspired minimalism with data clarity

**Core Principles:**
- Asymmetric two-column layout: sticky form panel left, scrollable results right
- Strict typographic hierarchy using weight and size contrast
- Teal/slate color system with bright green for positive results
- Generous whitespace to let data breathe

**Color Philosophy:** Off-white background (#f8fafc), deep slate (#0f172a) for headings, teal (#0d9488) as primary action color, emerald (#10b981) for result highlights. Clean, trustworthy, professional.

**Layout Paradigm:** Sticky left sidebar (40%) for all inputs, scrollable right panel (60%) for results, wind rose, map, and methodology breakdown. No full-page sections — everything visible at once.

**Signature Elements:**
- Thin teal left-border accent on active input groups
- Circular wind rose diagram using SVG/Canvas
- Numbered step indicators for the calculation methodology

**Interaction Philosophy:** Progressive disclosure — basic inputs visible first, advanced options expand on demand. Results update live as inputs change.

**Animation:** Smooth accordion expand for advanced settings. Wind rose rotates/animates on data load. Result cards fade in sequentially.

**Typography System:** Space Grotesk for headings (bold, geometric), Inter for body and labels. Tabular numbers for all data values.
</text>

## Approach 3: Field-Ready Mobile-First Tool
<response>
<text>
**Design Movement:** Material Design 3 / Field Engineering Tool

**Core Principles:**
- Mobile-first vertical flow optimized for tablet/phone use in the field
- High-contrast cards with rounded corners and strong shadows
- Forest green and earth tones reflecting the renewable energy context
- Large touch targets and clear CTAs

**Color Philosophy:** Forest green (#166534) as primary, warm cream (#fef9c3) as background, deep brown (#451a03) for text. Earthy, sustainable, approachable.

**Layout Paradigm:** Single-column vertical scroll with sticky "Calculate" CTA. Collapsible accordion sections for each input category.

**Signature Elements:**
- Leaf/wind iconography throughout
- Large result cards with oversized numbers
- Color-coded feasibility indicator (green/amber/red)

**Interaction Philosophy:** Wizard-style step-by-step input flow. One concept per screen. Clear progress indicator.

**Animation:** Slide-up transitions between steps. Bounce effect on result reveal.

**Typography System:** Nunito Sans (rounded, friendly) for all text. Extra-bold for results, regular for labels.
</text>
